앱 소개
독서실에서 알람 소리 없이도 타이머를 맞출 수 있게 해주는 단순한 앱 독서실에서 공부할때 사용하기 위해 개발


---
언어: Kotlin
버전 관리: Git
---

앱 기능
타이머 기능과 손전등 기능을 결합하여, 타이머의 시간이 끝나면 핸드폰의 손전등 기능이 켜졌다 꺼졌다 반복하게 됨. cancel버튼을 누를시 해제
해당 앱을 play store에 출시했었으며, 현재는 개발자 계정 이슈로 내려간 상태
